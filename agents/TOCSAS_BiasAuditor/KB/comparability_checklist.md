# Cross-Study Comparability Checklist for Preclinical Toxicology

## Purpose
Use this checklist when the TOCSAS swarm synthesizes findings across two or more preclinical toxicology studies. Each item that differs between studies represents a potential comparability risk that must be disclosed in section H of the output.

## Tier 1 — Critical Comparability Parameters (differences almost always limit valid comparison)
- [ ] Same compound (or same class/analogue — note if analogue)
- [ ] Same route of administration
- [ ] Same vehicle (or vehicle known to be pharmacologically inert for the endpoint of interest)
- [ ] Same species
- [ ] Same strain
- [ ] Same sex (or if both sexes, same sex-specific analysis available)
- [ ] Overlapping dose range
- [ ] Same duration class (e.g., both subchronic)

## Tier 2 — Important Comparability Parameters (differences require explicit disclosure)
- [ ] Same study type (GLP vs. non-GLP; regulatory guideline study vs. mechanistic)
- [ ] Same age at study start (within ±2 weeks for rodents)
- [ ] Same body weight range at study start
- [ ] Concurrent historical control data from comparable facility/time period
- [ ] Same histopathology grading system and scale
- [ ] Same pathologist or pathology peer review process
- [ ] Same clinical chemistry analyzer and reference method
- [ ] Toxicokinetic data available for both studies (allows AUC-based comparison)
- [ ] Same dosing schedule and frequency (once daily vs. BID vs. dietary admixture)

## Tier 3 — Recommended Comparability Parameters (differences noted but less likely to invalidate comparison)
- [ ] Same housing conditions (cage type, group size, LD cycle)
- [ ] Same diet
- [ ] Same formulation stability confirmation method
- [ ] Same statistical methods
- [ ] Same acclimatization period

## Comparability Risk Rating
After completing the checklist, assign a risk rating:
- **Low**: All Tier 1 items match; most Tier 2 items match.
- **Moderate**: All Tier 1 items match; ≥2 Tier 2 items differ with disclosed consequence.
- **High**: ≥1 Tier 1 item differs — comparison is exploratory only; conclusions must be heavily caveated.
- **Not valid**: Route or species differ without justification — direct comparison of findings is not supported.

## Common Confounders to Flag Explicitly
- Dietary admixture dosing vs. gavage: different peak plasma concentrations for same nominal dose.
- PEG-400 or DMSO vehicles at high concentrations: known to cause histopathological changes (renal tubular vacuolation, hepatic effects).
- Sprague-Dawley vs. Wistar rats: different background lesion rates for some organs.
- Young vs. adult animals: enzyme induction capacity and baseline organ weights differ.
