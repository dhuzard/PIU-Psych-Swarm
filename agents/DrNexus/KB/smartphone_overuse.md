# Smartphone Overuse — Orchestrator Reference

## Rapid Framing for Task Decomposition

When a task involves problematic smartphone use (PSU) or smartphone overuse:

| Sub-question | Best specialist(s) |
|---|---|
| Clinical features, instruments, comorbidity | ClinicalPsych |
| Prevalence, instrument heterogeneity, demographic patterns | EpiScope |
| Sleep disruption, attentional effects, neurobiological mechanisms | NeuroCogs |
| Interventions: app-based, digital detox, school-based, CBT adaptations | CarePath |
| Citation chaining (Billieux, Kwon, Ward, Lopez-Fernandez) | LitScout |

## Key Disambiguation Points

- **No formal diagnosis.** PSU / smartphone overuse has no ICD-11 or DSM-5 status.
- **Scale proliferation problem.** >50 instruments for PSU; cut-offs are not standardised.
  EpiScope must cite instrument + cut-off for all prevalence claims.
- **Duration ≠ disorder.** Many people use smartphones 6–8 hours/day for legitimate work
  and communication; functional impairment is the threshold.
- **Nomophobia is not PSU.** Fear of being without a phone is a separate, dimensional construct.

## Routing Heuristic

If task is about PSU and sleep → route NeuroCogs (sleep mechanisms) + CarePath (bedtime hygiene interventions) in parallel.
If task is about PSU and academic performance → route EpiScope + NeuroCogs in parallel.
If task asks for prevalence → route EpiScope with explicit instruction to name scale and cut-off.
If task is about treatment → route CarePath lead.

## Common Framing Errors to Prevent

- Do NOT allow the Journalist to state a prevalence figure without instrument + cut-off; reviewer will reject.
- Do NOT conflate PSU with general PIU or IGD findings; they may not share mechanisms.
- Do NOT present phone-free interventions ("digital detox") as evidence-based without noting the
  current absence of RCT evidence for these approaches.
