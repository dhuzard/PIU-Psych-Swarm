# Social Media Disorder — Orchestrator Reference

## Rapid Framing for Task Decomposition

When a task involves social media disorder (SMD) or problematic social media use:

| Sub-question | Best specialist(s) |
|---|---|
| Clinical features, comorbidity, diagnostic cautions | ClinicalPsych |
| Prevalence, instruments (BSMAS, SMDS), demographic patterns | EpiScope |
| Mechanisms: variable-ratio reinforcement, social reward, cognitive effects | NeuroCogs |
| Interventions: CBT, digital literacy, school-based programs | CarePath |
| Citation chaining (Andreassen, van den Eijnden, Valkenburg) | LitScout |

## Key Disambiguation Points

- **Not a recognised diagnosis.** SMD has no ICD-11 or DSM-5 status. Frame as research construct.
- **Passive vs. active use matters.** Mental health associations are driven primarily by passive
  consumption (scrolling, comparison), not active use (messaging, creating).
- **Platform dependency.** Most evidence comes from Facebook/Instagram-era studies; generalisability
  to TikTok, Snapchat, X is uncertain.
- **Adolescent females are the highest-risk group** in Western samples; male-dominated in some
  East Asian studies.

## Scope Boundaries

When routing an SMD task:
1. EpiScope must name instrument + cut-off for every prevalence figure.
2. NeuroCogs should flag whether cited neuroimaging studies used disorder criteria or heavy-use criteria.
3. CarePath should distinguish SMD-specific interventions from general PIU or IGD treatments.
4. Journalist must note that SMD is not a formal diagnosis.

## Routing Heuristic

If task asks about social media and mental health broadly → route NeuroCogs + EpiScope in parallel.
If task asks about treatment → route CarePath lead, CarePath + ClinicalPsych in parallel.
If task asks about adolescent-specific effects → route ClinicalPsych + NeuroCogs in parallel.
