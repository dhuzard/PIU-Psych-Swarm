# Code of Conduct

## Our Commitment

We are committed to making this project welcoming to researchers, clinicians, and developers of all backgrounds. Participation in this project means agreeing to treat everyone with respect and professionalism.

## Standards

**Expected behavior:**
- Use respectful, inclusive language in issues, pull requests, and discussions
- Be constructive when giving feedback on code or documentation
- Acknowledge when you are uncertain or wrong
- Focus criticism on ideas and implementations, not on individuals

**Unacceptable behavior:**
- Personal attacks, harassment, or discriminatory comments
- Sharing others' private information without consent
- Sustained disruption of discussions
- Any behavior that would be unwelcome in a professional research environment

## Scope

This code of conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any communication that references this project.

## Reporting

If you observe unacceptable behavior, open a private issue or contact the repository maintainer directly via GitHub. All reports will be handled confidentially.

## Enforcement

Maintainers may remove comments, close issues or pull requests, and temporarily or permanently exclude participants who violate these standards.
